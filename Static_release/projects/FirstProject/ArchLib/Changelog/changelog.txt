Список изменений Standard Peripherals Library.

--------------------------------------------------------------------------------

Об архиве Standard Peripherals Library:

В архиве SPL собраны соответсвующие SPL для различных МК. Полный список изменений необходимо смотреть в changelog файле на соответсвующий МК.

    - Исправлено подключение библитеки SPL в примерах проектов для IAR EW for RISC-V для МК К1986ВК025,MDR1206FI,MDR1206AFI.

Версии библиотек для МК в составе SPL:

    - SPL_MDR32FxQI   - Библиотека для МК К1986ВЕ92QI,К1986ВЕ1QI,К1901ВЦ1QI - версия 1.2.0
    - SPL_MDR1986VK01 - Библиотека для МК К1986ВК01GI                       - версия 1.4.0
    - SPL_MDR32VF0xI  - Библиотека для МК К1986ВК025,MDR1206FI,MDR1206AFI   - версия 0.1.0
    - SPL_MDR32F2xQI  - Библиотека для МК К1986ВК214,К1986ВК234             - версия 1.0
